## 🚀 Transforming NGO Operations with Technology

*A complete web-based NGO Management System developed during my internship at **SpaceECE India Foundation**. The platform digitizes NGO operations by integrating donor management, volunteer coordination, proposal management, donation tracking, and reporting into one centralized solution.*

</div>

# 📖 Overview

Traditional NGO operations often rely on paperwork and disconnected spreadsheets, making data management difficult and time-consuming.

The **NGO Management System** modernizes these workflows through an integrated digital platform that enables NGOs to efficiently manage donations, volunteers, fundraising campaigns, beneficiaries, proposals, and organizational records while improving transparency and operational efficiency.

# 🚀 Installation

### Clone Repository

```bash
git clone https://github.com/NUPUR32/NGO_Management_System.git
```

### Move to XAMPP

```
xampp/htdocs/
```

### Start Services

* Apache
* MySQL

### Create Database

```
ngo_management
```

### Import

```
database/ngo_db.sql
```

### Configure Database

```php
$host="localhost";
$user="root";
$password="";
$database="ngo_management";
```

### Run Project

```
http://localhost/NGO_Management_System
```
